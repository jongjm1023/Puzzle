# 후기

- 이종민
    
    1주일 안에 시간 역행과 맵 뒤집기가 가능한 3d 퍼즐 게임을 만드는게 사실상 불가능하다고 생각했었는데 어찌저찌 됬네요. 시간 역행 관련한 버그를 모두 수정한 현수에게 감사를 드립니다. 유니티 3d 관련한 지식을 정말 많이 습득한 한 주 였습니다.
    

- 김현수
    - 머릿속으로 생각했던 주요 기능과 오브젝트들이 실제로 작동하는게 너무 재밌었다.
    - 3d 플랫포머 게임의 버그 잡기가 상상 이상으로 까다로웠다. (특히 시간 역행)
    - 오브젝트 기획을 정밀한 기획서 없이 했는데, 종민이가 알잘딱으로 의도대로 구현해줘서 편했다.
    - 마찰이나 공기저항 수치 잡는건 여전히 잘 모르겠다….
